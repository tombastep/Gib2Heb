# Gib2Heb

Gibberish to Hebrew converter for Rhino 6, Created by Tom Melloul, April 2020.

# Installation

1.  Download "Rhino Plugin" folder's contents.
2.  Drag and drop RHP file (Plugin)
3.  Drag and Drop RUI file (Toolbar)
4.  You're good to go!
